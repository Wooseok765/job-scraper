Job Scraper Project

This project is a Python Flask web scraper that searches job postings and displays the results on a web page.

Project Description

The user enters a job keyword. The Flask application sends the keyword to the server, scrapes job information, and displays the search results.

Main Features

  • Search jobs by keyword
  • Scrape job title, company, location, and source
  • Display job results in an HTML page
  • Use Python, Flask, Requests, and BeautifulSoup
