Job Scraper Project

This project is a Python Flask web scraper that searches job postings and displays the results in a web page.

Project Description

The application allows users to enter a keyword, sends the keyword to the Flask server, scrapes job data from job websites, and displays the collected job results.

Main Features

  • Search jobs by keyword
  • Scrape job title, company, location, and source
  • Display search results in an HTML page
  • Use Flask, Requests, and BeautifulSoup
