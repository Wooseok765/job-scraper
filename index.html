Job Scraper Project

This is a Flask-based job scraper project.

Project Description

Users can search job keywords such as python, javascript, and java.

The app scrapes job results from:

  • Berlin Startup Jobs
  • Web3 Career
  • We Work Remotely

How to Run Locally

pip install flask requests beautifulsoup4
python app.py

Then open: 

http://127.0.0.1:5000

Note

GitHub Pages can host only static pages. Therefore, the Flask server must be run locally.
